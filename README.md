## Project Overview

PawHope is a non-profit organisation dedicated to rescuing, rehabilitating, and rehoming abandoned animals. The project aims to create a functional, responsive, and visually appealing website to showcase the organisation’s mission, facilitate adoptions, encourage volunteering and donations, and provide educational resources to the community.

The website will be the central hub for PawHope’s operations, integrating multiple pages with intuitive navigation, interactive features, and multimedia content.

---

## Website Goals and Objectives

- Raise awareness about animal rescue and welfare.
- Promote adoption of stray animals through an online gallery.
- Enable volunteers and donors to engage with the organisation.
- Educate the public about responsible pet ownership.

**Key Performance Indicators (KPIs):**

- 20% increase in adoption enquiries within 6 months.
- 50+ new volunteers and sponsors registered in the first year.
- 30% growth in online engagement and traffic.

## Key Features and Functionality

| Page                 | Features                                                                                                                 |
| -------------------- | ------------------------------------------------------------------------------------------------------------------------ |
| **Homepage**         | Hero section with call-to-action (“Adopt Today”), latest rescue stories, featured animals                                |
| **About Us**         | Organisation history, mission, vision, team introduction                                                                 |
| **Adopt**            | Gallery of adoptable animals with filters (species, age, size), profile pages with photos, descriptions, enquiry buttons |
| **Accessories**      | Merchandise for fundraising (collars, leashes, T-shirts, mugs)                                                           |
| **Get Involved**     | Volunteer registration, sponsorship packages, donation options                                                           |
| **Contact**          | Contact details, multiple rescue points on Google Maps, enquiry form                                                     |
| **Blog / Resources** | Articles on pet care, vaccination schedules, and educational campaigns                                                   |
